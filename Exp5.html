1Ô∏è‚É£ Setting the Viewport

The viewport meta tag is essential for responsive design. It gives the browser instructions on how to control the page's dimensions and scaling.

<!-- Essential viewport meta tag for responsive design --> <meta name="viewport" content="width=device-width, initial-scale=1.0">

What it does:

  • width=device-width - Sets page width to follow screen width
  • initial-scale=1.0 - Sets initial zoom level when page loads

2Ô∏è‚É£ Responsive Images

Images that scale nicely to fit any browser size using CSS properties.

width: 100%

Image scales up & down
<img src="image.jpg" style="width:100%;">

Image will be responsive and scale up and down

max-width: 100%

Never larger than original
<img src="image.jpg" style="max-width:100%;height:auto;">

Image scales down but never larger than original size

4Ô∏è‚É£ Responsive Text Size

Text size can be set with "vw" unit (viewport width), so text size follows browser window size.

Hello World

This text uses font-size: 6vw - resize your browser to see it change!

<!-- Responsive text using viewport width --> <h1 style="font-size:10vw">Hello World</h1>

Viewport Units:

  • vw - Viewport Width (1vw = 1% of viewport width)
  • vh - Viewport Height (1vh = 1% of viewport height)
  • vmin - Smaller of vw or vh
  • vmax - Larger of vw or vh

/* CSS for responsive three-column layout */ .left, .right { float: left; width: 20%; /* Default width */ } .main { float: left; width: 60%; /* Default width */ } /* Media query for screens 800px and smaller */ @media screen and (max-width: 800px) { .left, .main, .right { width: 100%; /* Full width on mobile */ } }

‚úÖ Responsive Design Best Practices

üì± Mobile First

Start designing for mobile, then enhance for larger screens using min-width media queries

üéØ Content Priority

Most important content should be visible on smallest screens first

‚ö° Performance

Use appropriate image sizes and optimize for faster mobile loading

üëÜ Touch Friendly

Buttons and links should be large enough for finger taps (44px minimum)

üîç Testing

üìê Flexible Units
